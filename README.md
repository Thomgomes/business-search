## Rodando localmente

Clone o projeto

```bash
  git clone ...
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npx expo start

  ou

  expo start
  
  ou 

  npm start
```
